# Poojak AI Factory Native v9

Made by Poojak Doshi.

This is the **native Android Gradle version**. It is not the 17 KB WebView wrapper.

## Features
- Native Android app UI
- OpenAI / Gemini / Ollama / LM Studio / Custom endpoint brain router
- Natural-language AI maker: "make an AI for digital marketing"
- Agent library in memory
- Short direct replies
- Child APK build plan screen
- Real Android dependencies: Material, OkHttp, Room, WorkManager, Markwon, ML Kit, TensorFlow Lite

## Build
Use GitHub Actions `Build Native APK`, Android Studio, or a PC/VPS with Android SDK + Gradle.

Do not paste API keys into chat. Put keys inside the app Brain Settings.

## Note
APK size will be natural from real Android libraries. Exact size depends on Gradle build output and device ABI packaging.
